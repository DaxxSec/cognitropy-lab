# Doctrine References — Curated Standards Bibliography

The agent cites these inline in scoring sheets, briefs, and after-action reports. They are the audit trail.

## Risk-Methodology Standards
- **ISO 31000:2018 — Risk Management Guidelines** — establishes the *establish context → identify → analyse → evaluate → treat → monitor & review* cycle this workspace's workflows mirror. <https://www.iso.org/iso-31000-risk-management.html>
- **ISO/IEC 31010:2019 — Risk Assessment Techniques** — catalogs the 5×5 matrix among many techniques; useful for justifying methodology choice.
- **MIL-STD-882E — System Safety** — origin standard for the modern 5×5 matrix in defense contexts; tier definitions translate cleanly to protective ops.
- **NIST SP 800-30 Rev 1 — Guide for Conducting Risk Assessments** — useful for structuring the threat-baseline section's actor / capability / intent decomposition.

## Protective-Operations Doctrine
- **U.S. Department of State, Bureau of Diplomatic Security — High-Threat Protection (HTP) program** (public summaries; full doctrine is restricted). The dominant U.S. methodology for Tier 1 / non-permissive environments.
- **U.S. Secret Service — Protective Methodology overview** (public-facing). Source for "concentric rings of protection" framing and curb-line operations.
- **U.S. Marshals Service — Witness Security (WITSEC)** — relevant for protected-witness movement workflows.
- **NPSA (National Protective Security Authority, UK)** — strong public guidance on hostile vehicle mitigation (HVM), reconnaissance indicators, crowded-place protection. <https://www.npsa.gov.uk/>
- **Royal Canadian Mounted Police — Prime Minister's Protective Detail / Protective Policing Services** (open posture documents).
- **ASIS International PSC.1 — Executive Protection Standard** — corporate-EP risk management; aligns with ISO 31000.
- **ASIS International PSC.2 — Quality of Private Security Operations** — useful for vetting contracted detail.

## Threat-Baseline Inputs
- **OSAC (Overseas Security Advisory Council)** — daily summaries, AOR threat assessments. <https://www.osac.gov/>
- **U.S. State Department Travel Advisories** — country-tier risk pegs. <https://travel.state.gov/>
- **UK FCDO Travel Advice** — open. <https://www.gov.uk/foreign-travel-advice>
- **GAC Canada / DFAT Australia / Auswärtiges Amt** — equivalent host-government advisories.
- **ACLED — Armed Conflict Location & Event Data** — political violence and protest dataset for AOR conflict density. <https://acleddata.com/>
- **GDELT** — global event database; high noise but useful for trend signals.
- **INTERPOL — Project Geiger** and travel-threat resources.

## Hostile-Vehicle Mitigation (HVM)
- **PAS 68 / PAS 170 (UK)** — hostile vehicle mitigation product-and-installation standards.
- **ASTM F2656 (US)** — vehicle-impact rating standard for barriers.
- **NPSA HVM Guidance** — open guidance on stand-off, layout, and crowded-place protection.

## Vehicle Hardening
- **VPAM BRV 2009 / 2010** — European standard for ballistic and explosion-resistant vehicles (VR / ERV ratings).
- **NIJ Standard-0108 / 0101** — U.S. ballistic-resistance standards for armor.
- B6 / B7 informally maps to VPAM VR7 / VR9.

## Incident Command & Liaison
- **NIMS / ICS — National Incident Management System (US)** — for interfacing with U.S. responders.
- **Gold-Silver-Bronze command (UK)** — host-nation interfacing for UK ops.
- **Sphere of Influence frameworks** at host-nation MOI / MFA / police levels.
