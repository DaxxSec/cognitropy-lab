# 32-Point Per-Segment Route Survey Checklist

Use during physical drive of a candidate route. One sheet per segment. Tick as recorded; flag with `*` if the answer drives a dominant hazard tag.

## Geometry
- [ ] 1. Start lat/long captured (5 decimal places)
- [ ] 2. End lat/long captured
- [ ] 3. Length in meters (approx)
- [ ] 4. Lane count in motorcade direction
- [ ] 5. Divided / undivided / one-way
- [ ] 6. Median width if divided
- [ ] 7. Speed regime: highway / urban / dense-urban / parking / tunnel / underground

## Sightlines & Standoff
- [ ] 8. Minimum line-of-sight standoff to any plausible firing position (meters)
- [ ] 9. Buildings overlooking the segment with windows that open
- [ ] 10. Rooftop access to overlooking buildings (visible / hardened / unknown)
- [ ] 11. Tunnels, underpasses, overpasses in segment

## Choke & Egress
- [ ] 12. Chokepoint Y/N (and reason — bridge / lane reduction / construction / parked vehicles)
- [ ] 13. Ingress/egress count — alternate exits within segment
- [ ] 14. Median crossings possible Y/N
- [ ] 15. Estimated time-to-clear if blocked at start of segment

## Crowd & Population
- [ ] 16. Crowd density at planned T-0 (persons per 100 m²; estimate)
- [ ] 17. Public event overlap (parade, protest, sports)
- [ ] 18. Pedestrian density / sidewalk activity
- [ ] 19. School / hospital / public-attractant venues abutting segment

## Jurisdiction & Liaison
- [ ] 20. Police district / municipality
- [ ] 21. Liaison reachable on encrypted host-nation comms Y/N
- [ ] 22. Recent host-nation security operations in segment (last 30 d)
- [ ] 23. Cellular coverage confirmed Y/N

## Threat Indicators (advance team observation)
- [ ] 24. Pre-positioned vehicles flagged Y/N
- [ ] 25. Surveillance indicators flagged Y/N (loiterers, repeat-pass vehicles)
- [ ] 26. Disturbed pavement / unscheduled roadwork Y/N
- [ ] 27. Abandoned objects (bag, crate, moped) at chokepoints Y/N
- [ ] 28. Anomalous local behavior (early-closing shops, empty street)

## Hazards & Mitigations
- [ ] 29. Dominant hazards from typology — list 1–3
- [ ] 30. Mitigations available on this segment for this leg (list)
- [ ] 31. MEDEVAC route from this segment to nearest trauma facility (record)
- [ ] 32. Photos taken — file paths recorded (kept offline)
