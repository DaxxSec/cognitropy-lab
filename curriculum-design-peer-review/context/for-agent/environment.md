# Environment

*Populated by `/onboard`. Defaults shown below.*

## Working Environment
- **Platform:** Claude agent workspace with filesystem access
- **Primary Output Format:** Markdown (.md)
- **Version Control:** Git (recommended for tracking curriculum iterations)

## Typical User Environment
- Learning Management System (LMS): Canvas, Blackboard, Moodle, Google Classroom
- Document tools: Google Docs, Microsoft Word
- Standards databases: State/national standards portals

## File Conventions
- Unit plans: `outputs/unit-[topic]-v[version].md`
- Reviews: `outputs/review-[doc-name]-[date].md`
- Rubrics: `outputs/rubric-[assessment-name].md`
- Alignment matrices: `outputs/alignment-[subject]-[framework].md`
- Session logs: `work-log/[YYYY-MM-DD]-session.md`
