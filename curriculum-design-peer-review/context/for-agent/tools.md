# Tools

## Core Tools (Built-in)
- **Filesystem** — Read/write curriculum documents, manage workspace files
- **Markdown rendering** — Format unit plans, rubrics, alignment matrices
- **Search** — Find standards, reference materials in resources/

## Recommended MCP Servers
- **Filesystem MCP** — Direct file management for curriculum documents
- **Google Docs MCP** — Collaborate on shared curriculum drafts in Google Workspace
- **Notion MCP** — Use database views for curriculum mapping and tracking
- **Git MCP** — Version control curriculum iterations and track review cycles
- **Google Sheets MCP** — Build and maintain standards alignment matrices

## External Standards Resources
- Common Core State Standards: corestandards.org
- Next Generation Science Standards: nextgenscience.org
- ISTE Standards: iste.org/standards
- Webb's DOK resources: Various educational portals
