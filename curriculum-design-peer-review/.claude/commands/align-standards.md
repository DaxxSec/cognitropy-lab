# Align Standards

Map learning objectives to one or more standards frameworks.

## Input
1. A list of learning objectives (from a unit plan or provided directly)
2. The target standards framework(s)

## Process

1. **Parse objectives** — Extract the verb, content, and conditions from each objective
2. **Identify applicable standards** — Match each objective to relevant standards by content area and cognitive level
3. **Build alignment matrix** — Create a table showing:
   | Objective | Bloom's Level | DOK Level | Standard(s) | Coverage |
4. **Gap analysis** — Identify:
   - Standards required by the course/unit that no objective addresses
   - Objectives that don't map to any standard (may still be valid — flag for review)
5. **Recommendations** — Suggest additional objectives to close gaps, or note where existing objectives could be revised to better align

## Output
Save alignment matrix to `outputs/alignment-[subject]-[framework].md`
Present summary to user with gap analysis and recommendations.
