# Bloom's Objective Writer

Write a set of learning objectives for a curriculum unit using Bloom's Revised Taxonomy.

## Prompt

Given the following unit details:
- **Subject:** [subject area]
- **Topic:** [specific topic]
- **Grade/Audience Level:** [level]
- **Unit Duration:** [number of sessions/weeks]

Write 5-8 learning objectives that:
1. Use measurable verbs from Bloom's Revised Taxonomy
2. Progress from lower-order (Remember/Understand) to higher-order (Analyze/Evaluate/Create)
3. Follow the format: "Students will be able to [verb] [what] [under what conditions/to what standard]"
4. Are specific enough to be directly assessed
5. Cover both factual/conceptual knowledge and procedural/metacognitive skills

For each objective, note:
- The Bloom's level (1-6)
- The knowledge dimension (Factual, Conceptual, Procedural, Metacognitive)
- A suggested assessment method
