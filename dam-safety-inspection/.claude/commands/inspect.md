# Inspect

Generate a comprehensive inspection checklist for a dam.

## Usage
Provide the following dam information:
- **Dam type**: embankment (earth/rockfill), concrete gravity, arch, buttress, RCC
- **Key features**: spillway type, outlet works, instrumentation installed
- **Hazard classification**: high, significant, or low
- **Known concerns**: any existing deficiencies or areas of special attention

## Process
1. I will select the appropriate failure mode set for your dam type
2. Generate a feature-by-feature inspection checklist (crest, upstream face, downstream face, abutments, spillway, outlet works, toe/seepage collection, instrumentation)
3. Include specific visual indicators to look for at each feature
4. Note which observations relate to which potential failure modes
5. Add measurement/data collection tasks alongside visual checks

## Output
A structured, printable inspection checklist suitable for field use, with severity rating guidance for each observation category.
