# Explain Bayesian Results to Stakeholders

Use this prompt when you need to translate Bayesian analysis outputs into language that non-technical stakeholders (project managers, elected officials, regulatory reviewers) can understand.

## Prompt

I just completed a Bayesian analysis with the following results:
- [Paste posterior summary or describe key findings]
- [Include any credible intervals or probability statements]

My audience is: [describe — city council, FEMA reviewer, utility board, project manager]

Please:
1. Translate the statistical results into plain language
2. Create a 1-paragraph executive summary focusing on the decision implications
3. Suggest 1-2 simple visualizations that communicate the key takeaway
4. Identify the "so what" — what action should this analysis drive?
5. Anticipate 3 questions the audience is likely to ask and draft answers
