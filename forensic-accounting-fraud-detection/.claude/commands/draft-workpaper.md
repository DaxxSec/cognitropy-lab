# /draft-workpaper — Generate Investigation Workpaper

I'll structure your findings into a professional investigation workpaper following ACFE documentation standards.

## What to Provide
Tell me about:
1. **Matter**: Case name/number, investigation trigger
2. **Scope**: What was examined, data sources, date ranges
3. **Findings**: What anomalies or concerns were identified (can be rough notes)
4. **Evidence**: Transaction IDs, document references, amounts
5. **Examiner**: Name(s) and date(s) of work

## Workpaper Structure I'll Generate
1. **Cover Page** — matter, examiner, date, scope statement
2. **Executive Summary** — top findings, estimated dollar exposure, recommended actions
3. **Methodology** — tests performed, populations, tools used
4. **Findings Detail** (one section per finding):
   - Finding description and evidence citations
   - Risk classification: Confirmed / Probable / Possible / Indicator
   - Dollar exposure (if quantifiable)
   - Recommendation
5. **Limitations** — what was not examined, reliance on representations, data gaps
6. **Standard Disclaimer** — analytical assistance only, not professional attestation

## Output Format
- Formatted Markdown (ready to paste into Word/Google Docs)
- Optionally: plain text or structured JSON for tool import

Describe the matter and your findings, and I'll draft the workpaper.
