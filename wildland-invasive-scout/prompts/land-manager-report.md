# Prompt: Land Manager Notification Report Generator

**When to use:** When you've detected a new invasive incursion or significant ecological concern that needs to be reported to a land manager, extension office, or conservation agency.

**Paste this prompt and fill in the brackets:**

---

I need to generate a formal notification report to send to [land manager / county extension office / state invasive species program / USDA APHIS].

Here's what I found:

**Detection Summary:**
- Species (or suspected species): [  ]
- Detection date: [  ]
- Location (GPS or description): [  ]
- Habitat: [  ]
- Abundance at time of detection: [isolated individual / small patch / large infestation]
- Photos: [Y/N]
- My confidence in the ID: [low / medium / high — describe why]

**Supporting Context:**
- My background/role: [landowner / recreation user / guided / professional surveyor]
- How I found it: [during foraging / bushcraft outing / survey / other]
- Previous history of this area: [never seen this before / have been coming here X years / other]
- Any known spread vectors nearby? [trails / streams / roads / wildlife corridors]

**What I've already done:**
- Photographed: [Y/N]
- iNaturalist submission: [Y/N — observation link if available]
- EDDMapS report: [Y/N]
- Any management actions taken: [none / documented only / marked GPS / other]

Please generate:
1. A professional notification email to [agency type] with the key detection details
2. A concise site description and location narrative
3. Recommended immediate actions I should convey to them
4. Questions I should ask them about follow-up management
5. Any state-specific regulatory notes I should be aware of for this species

Format as a ready-to-send email, professional but accessible tone.
