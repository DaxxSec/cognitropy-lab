# Prompt: Unknown Species Description Intake

**When to use:** When you've found something you don't recognize and want a systematic ID attempt. Fill in as much as you can observe.

**Paste this prompt and fill in what you can observe:**

---

I've found an unknown [plant / mushroom / animal / insect / other] and need help identifying it. Here's everything I can observe:

**Location & Habitat:**
- State/Region: [  ]
- Habitat type: [forest / forest edge / stream bank / open field / disturbed area / wetland / other]
- What it's growing near / associated with: [  ]
- Sun/shade: [full sun / partial shade / deep shade]

**Growth Form:**
- Type: [tree / shrub / vine / herbaceous / grass / sedge / fern / mushroom / animal]
- Approximate size/height: [  ]
- Growing: [upright / trailing / climbing / rosette / other]

**Leaf/Cap/Frond (if applicable):**
- Shape: [  ]
- Leaf edge: [smooth / toothed / lobed / deeply cut]
- Leaf arrangement on stem: [opposite / alternate / whorled / basal rosette]
- Top surface: color, shiny/matte, smooth/hairy?
- Bottom surface: color, veins visible, any special features?
- Smell when crushed: [  ]

**Stem/Bark/Stalk:**
- Color: [  ]
- Texture: [smooth / hairy / waxy / ridged / thorny]
- Cross section if broken: [round / square / hollow / solid]
- Any notable features: [  ]

**Flowers/Fruit/Spores (if visible):**
- Flower color and structure: [  ]
- Fruit type: [berry / nut / pod / capsule / other]
- Fruit color: [  ]
- Seed dispersal method apparent: [wind / animal / water / explosive]

**Root (if visible):**
- Type: [  ]
- Color/smell: [  ]

**Photos available:** [Y/N — describe what angles you have]

**Season/Phenology state:** [new growth / mature / flowering / fruiting / dormant / other]

**My best guess (if any):** [  ]

**Why I'm concerned:** [just curious / looks like it might be invasive / foraging question / camp near it]

---

Please:
1. Give me your best probable ID with confidence percentage
2. List the features that match and any that concern you
3. Identify the top dangerous lookalikes and how to rule them out
4. Tell me its invasive status for my region
5. Recommend what I should do next
