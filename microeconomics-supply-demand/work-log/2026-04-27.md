## What Got Built Today

A bespoke Claude Agent Workspace specialized for **microeconomic supply-and-demand analysis using calibrated risk-scoring matrices**. Bridges two normally-disconnected disciplines:

- Microeconomics (S/D, elasticity, surplus, comparative statics, Hicks-Slutsky/BLP/Tirole roots)
- Operational risk management (ISO 31000, IEC 31010, COSO ERM, NIST 800-30, AIAG-VDA FMEA)

The workspace's distinctive move is using elasticity as the *bridge* — every matrix cell is anchored to an elasticity-derived likelihood and impact, never gut-feel.

## Notes for Future Sessions

- The workspace deliberately refuses cross-firm pricing coordination — Sherman §1 / Article 101 hard line baked into `constraints.md` and `cross-elasticity-screen.md`.
- The L × I × D (FMEA) mode is supported but optional; default is L × I (RPN) on a 5×5.
- All elasticity priors are cited; the agent is expected to refuse "AI-detected" elasticity values not traceable to either user data or a literature source.
- First action for any new user: run `/onboard` then immediately `/calibrate-scale` — the workspace refuses to score uncalibrated.
