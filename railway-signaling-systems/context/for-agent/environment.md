# Environment: Railway Signaling Systems Workspace

## Learner Environment Profile

*Populate via `/onboard`. Default assumptions until then:*

**Work Context:** Network Rail or TOC/contractor apprenticeship in the UK
**Study Mode:** Blended — workplace learning with periodic classroom/computer-based assessment
**Primary Learning Goals:** Progress through NVQ competency units, prepare for IRSE Licence application
**Available Resources:** Network Rail standards (SPARK access), employer training center, hands-on lineside work

## Notes for Signal Mentor

- The learner works with live safety-critical infrastructure. Always reinforce the principle that uncertainty = stop, escalate, do not proceed.
- UK-specific standards and Network Rail practices are the default frame of reference unless the learner states otherwise.
- Apprentices may have varying mathematical backgrounds — check before assuming calculus-level AC circuit theory is accessible.
- Some learners are mature apprentices (career changers with prior engineering experience) — `/onboard` should probe for transferable knowledge.
- Heritage railway volunteers are a secondary user type — they operate under different (lighter) regulatory regimes but the physical principles are identical.
