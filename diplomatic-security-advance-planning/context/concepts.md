# Diplomatic Security Advance Planning — Core Concepts

Background the agent should read before acting on advance-planning, venue-survey, contingency-tree, or AAR tasks. Optimised for fast recall, not exhaustive theory.

## Advance-Planning Fundamentals

### What advance planning produces

A documented operational picture for a protectee visit covering: itinerary expansion, threat-tier-per-increment, venue surveys, route reconnaissance, motorcade configurations, host-nation coordination, contingency planning, communications redundancy, OPSEC handling. Each artifact carries an owner, a protocol reference, a sign-off, and an audit trail.

### Pre-trip phases (T-Δ days from arrival)

| Phase | Days | Activities |
|---|---|---|
| **T-30 to T-14** | Strategic advance | Embassy notification, host-nation diplomatic clearance, draft itinerary, intel feed activation, embassy DCM coordination |
| **T-14 to T-7** | Tactical advance | Venue surveys, route recons begin, host-nation MOI liaison, motorcade config draft, contingency tree draft, comms infrastructure verification |
| **T-7 to T-1** | Final advance | In-person recon completion, rehearsal, dead-zone testing, daily TBA briefs, motorcade rehearsal, contingency walkthrough |
| **T-0** | Activation | Detail handover, threat picture lock, deviation log open |

### In-trip + post-trip

| Phase | Activities |
|---|---|
| **Visit window** | Continuous deviation logging, twice-daily TBA briefs, real-time contingency-tree consultation if events develop |
| **Post-trip (T+72h)** | AAR within 72 hours, protocol revision proposals, archive trip artifacts per OPSEC handling |

## Threat Taxonomy

### Threat tiers (notional 4-tier scale)

| Tier | Profile | Example posture |
|---|---|---|
| **T1** | Low — friendly nation, no specific threat reporting | Low-profile 3-vehicle, no CAT, light host-nation police |
| **T2** | Standard — routine threat, generic regional concerns | 5-vehicle, light CAT, standard host-nation police |
| **T3** | Elevated — specific threat reporting OR high-civilian-density venues OR active regional issues | 7-vehicle, CAT, enhanced host-nation police, motorcycle leapfrog |
| **T4** | High — direct credible threat, conflict environment, post-attack response | Full motorcade with decoy, dedicated CAT, military augmentation, helicopter coverage if available |

Actual tier scales vary by service (DS DSS, USSS, RSO, contracted EP). Use the practice's documented scale; the principle is consistent.

### Threat categories

- **Direct threat** — specific intelligence indicating a planned attack on the protectee
- **Indirect threat** — generalised threat in jurisdiction (terrorism, civil unrest, political crisis)
- **Environmental threat** — context that elevates baseline (election cycle, anniversary, sectarian flare)
- **Tactical threat** — TTP-relevant intelligence (new IED design, new attack-pattern, new VBIED profile)

### Attack vectors

- **Small-arms attack (SAF)** — pistol, rifle, sub-machine gun from concealment or vehicle
- **Improvised explosive device (IED)** — pressure-plate, command-wire, RC-trigger, magnet
- **Vehicle-borne IED (VBIED)** — suicide-driven or remote-detonation
- **Magnetic / drop-and-flee IED** — placed on motorcade vehicle at chokepoint
- **Hostile vehicle (ramming)** — large vehicle used as attack vector against motorcade or venue
- **Civil disturbance escalation** — protest crowd becomes violent, traps motorcade
- **Kidnapping attempt** — ambush + extract; rare for protected principals but high-impact
- **Cyber / influence** — not direct kinetic but can degrade operational posture (comms shutdown, fake intel)

## Motorcade Doctrine

### Standard vehicle roles

| Role | Position | Function |
|---|---|---|
| **Lead** | First vehicle | Scout ahead, set pace, intersection management |
| **Control** | Vehicle 2 (varies by config) | Detail leader, comms hub, route decision authority |
| **VIP** | Vehicle 3 (typical) | Protectee, armoured, full detail agents |
| **Follow** | Vehicle 4 | Backup VIP if primary disabled, additional agents |
| **CAT** (Counter-Assault Team) | Vehicle 5 or trailing | Tactical response to direct attack, dismounts if needed |
| **Decoy** | Variable | Visually similar to VIP, decoys hostile observation |
| **Medical** | Co-located | Onboard medical capability |

### Movement protocols

- **Speed band** — set per route topology, never exceeds host-nation legal max except in evac
- **Spacing** — 30-50 m typical urban; expanded on highway; collapsed in chokepoints
- **Intersection management** — motorcycle leapfrog (host-nation officer pre-positions, holds traffic, releases after motorcade); host-nation holds (uniformed officers pre-position, hold traffic, release on cue)
- **Lane discipline** — typically left lane in right-hand-drive countries (right in LHD), avoid overtaking on inside
- **Communication frequencies** — primary channel for movement, alternate for emergencies, encrypted for sensitive traffic

## Venue-Survey Methodology

### Survey categories

| Venue type | Key sections |
|---|---|
| **Private residence** | Perimeter, neighbours, vendor/staff access, family movements, mailroom |
| **Hotel** | Floor selection, freight elevators, housekeeping, room-service vector, parking-garage egress, conference space |
| **Official building** | Inner cordon, accreditation, building security, holding rooms, motorcade staging, evacuation routes |
| **Public event** | Stage standoff, crowd-screening, perimeter, photo-line management, vendor stalls in proximity |
| **Religious institution** | Worship space, congregation movement, religious-protocol constraints, holy-day calendar |
| **Restaurant** | Kitchen access, staff vetting, vendor delivery times, alternative entrance/egress |
| **Transit point (airport/station)** | Terminal access, baggage handling, holding lounge, runway/platform access |

### Survey standards

- Conducted in person
- Photo-documented (subject to OPSEC)
- Signed off by advance agent + RSO + venue POC
- Re-surveyed every trip (no "previously surveyed" exemption)
- Threat-tier overlay applied (elevated tiers add sections like TSCM, counter-surveillance sweep)

## Communications Doctrine

### Channel hierarchy

| Tier | Channel | Use case |
|---|---|---|
| **Primary** | P25 / Tetra encrypted radio | Routine detail comms |
| **Alternate** | GSM / 4G phones (encrypted apps) | Backup for primary, host-nation liaison |
| **Emergency** | Satellite phone (Iridium, Inmarsat) | When terrestrial fails |
| **Manual** | Runner protocol, hand signals | Dead zones, comms loss |

### OPSEC posture

- Encrypted at the appropriate classification
- Key management documented; rotation per trip
- Radio discipline (no protectee identification in clear, code names)
- Open-channel announcements minimised

## Common Failure Modes

- **Protocol drift over time** — survey shortcut, recon truncation, AAR skipped — accumulates invisibly
- **Single-source threat reporting acted on without corroboration** — drives over-or-under-reaction
- **Ambiguous decision authority** in contingency branches — pauses response when speed matters
- **Comms dead zones discovered in moment** — should have been mapped + mitigated pre-trip
- **Host-nation coordination assumed rather than confirmed** — "we always do it this way" without current-trip verification
- **AAR skipped due to fatigue / next-trip pressure** — surfaces nothing; same gaps recur next trip
- **OPSEC failure in pre-trip planning artifacts** — protectee schedule leakage, venue specifics disclosure, motorcade-config exposure

## Operating Constraints

- **Host-nation sovereignty** — operations subject to host-nation consent and SOFA/BIA/MOU
- **Use-of-force** — standing rules vary by host nation and mission; documented in advance plan, not re-authored in moment
- **Privacy / OPSEC** — outputs sensitive; handle per practice's classification and access-control regime
- **Contractor scope** — contracted EP teams operate within contract scope; out-of-scope escalated, not absorbed
