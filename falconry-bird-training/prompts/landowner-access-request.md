# Landowner Access Request

## Purpose

Use to draft a courteous request for permission to hunt a landowner's ground with a trained hawk, and to capture the agreed terms. Falconry's low-impact, no-firearm nature is the lead. Pairs with `/landowner-access`.

## Prompt Template

```
You are helping me request permission to fly my hawk on a landowner's property.

- **Landowner / contact:** [name; owner or tenant?]
- **Property:** [location, rough acreage, boundaries]
- **Quarry present:** [rabbit / squirrel / game birds]
- **When I'd hunt:** [season, days, rough frequency]
- **What I offer:** [e.g. rabbit/squirrel control, sharing the field respectfully]
- **My details:** [name, falconry permit, contact, insurance if any]

Please draft a friendly, brief request that explains what falconry is (a single
trained bird, NO firearms, very low impact), asks for permission for the stated
dates, acknowledges liability and respect for the land, offers reciprocal value,
and requests a short written reply. Then give me a one-page terms record to fill in
(boundaries, no-go zones, gates/parking, livestock/dog notes).
```

## Expected Output

- A short, warm access-request message emphasizing low impact and no firearms.
- A clear ask for the season/dates and a request for written permission.
- A one-page terms record template (boundaries, no-go zones, access, conditions).
- A reminder to send an end-of-season thank-you to keep access for next year.
