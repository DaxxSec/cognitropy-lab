# Sponsor Quarterly Update

## Purpose

Use to write the recurring progress update an apprentice sends their sponsoring falconer. Turns the weight/training log into a readable account plus explicit questions. Pairs with `/sponsor-report`.

## Prompt Template

```
You are helping me write a progress update to my falconry sponsor.

- **Reporting period:** [e.g. Mar–May 2026]
- **Bird:** [species, source, age, band #]
- **Flying-weight band & current response:** [e.g. 1180–1210 g, keen and recalling well]
- **Flights this period:** [creance / free flights, distances, frequency]
- **Milestones reached:** [first creance flight, first free flight, first slip on quarry, first kill]
- **Equipment / husbandry changes:** [new jesses, telemetry, mews change, diet]
- **Concerns:** [any setbacks, weight wobble, behavior, facility]
- **Questions for you:** [the specific things I want guidance on]

Please draft a clear, humble update that leads with progress, is honest about
setbacks, and ends with my numbered questions. Friendly but organized.
```

## Expected Output

- A short narrative of the period's progress, weight band, and response.
- A milestones list and any equipment/husbandry changes.
- An honest concerns section (setbacks surfaced, not buried).
- A numbered list of specific questions for the sponsor.
