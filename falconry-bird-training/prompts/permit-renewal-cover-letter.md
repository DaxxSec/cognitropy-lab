# Permit Renewal Cover Letter

## Purpose

Use when filing a falconry permit application or annual renewal with your state wildlife agency and you want a clear, professional cover letter that summarizes the program and any changes since last cycle. Pairs with `/permit-packet`.

## Prompt Template

```
You are helping me draft a cover letter for my falconry permit [application / renewal] to my state wildlife agency.

- **Permit class:** [apprentice / general / master]
- **State agency & coordinator:** [name]
- **My permit number(s):** [federal / state]
- **Current birds:** [species, source, band numbers]
- **Species requested (if changing):** [species]
- **Sponsor (apprentices):** [name, class, permit #]
- **Facility:** [mews + weathering summary; note any changes since last cycle]
- **Changes since last renewal:** [new bird, facility upgrade, address change, none]
- **Filing deadline:** [date]

Please draft a concise, respectful cover letter that summarizes the program, notes
what (if anything) changed, confirms facility/inspection readiness, and requests
processing before the deadline. Keep it to one page.
```

## Expected Output

- A one-page cover letter addressed to the state coordinator.
- Clear statement of class, birds, species request, and any changes.
- A line confirming facility/inspection readiness.
- A polite deadline reference and contact details.
