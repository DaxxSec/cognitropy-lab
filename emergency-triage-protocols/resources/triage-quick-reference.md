# Triage Protocol Quick Reference — Decision Trees

## START (Simple Triage and Rapid Treatment)

**Use for:** Adults in mass casualty scenes | Assessment time: ~60 seconds

```
PATIENT ENCOUNTERED

STEP 1: RESPIRATION
├─ RR < 10 OR > 30? → IMMEDIATE (Red) 🔴
│  └─ STOP — Patient is Immediate
│
└─ RR normal (10–30)? → Continue to Step 2

        ↓
STEP 2: PERFUSION (Radial Pulse + Capillary Refill)
├─ No radial pulse present? → IMMEDIATE (Red) 🔴
│  └─ STOP
│
├─ CRT > 2 seconds (with radial pulse)? → IMMEDIATE (Red) 🔴
│  └─ STOP
│
└─ Radial pulse present AND CRT ≤ 2 sec? → Continue to Step 3

        ↓
STEP 3: MENTAL STATUS
├─ Cannot follow simple commands? → IMMEDIATE (Red) 🔴
│  └─ STOP
│
└─ Able to follow commands? → DELAYED (Yellow) 🟡

SPECIAL RULES:
• Walking patient at scene? → Likely Minor (Green) but still assess all 3 steps
• Not breathing after airway positioning? → EXPECTANT (Black) ⚫
  (Only if no hope of salvage; otherwise attempt resuscitation)
```

## JumpSTART (Pediatric, Ages 1–8)

**Use for:** Children 1–8 years | Modifications to START

```
PATIENT (AGE 1–8) ENCOUNTERED

STEP 1: RESPIRATION
├─ RR < 15 OR > 40? → IMMEDIATE (Red) 🔴
│  └─ STOP
│
└─ RR normal (15–40)? → Continue to Step 2

        ↓
STEP 2: PERFUSION (Brachial/Femoral Pulse Assessment)
├─ No pulse palpable? → IMMEDIATE (Red) 🔴
│  └─ STOP
│
├─ Pulse WEAK or THREADY? → IMMEDIATE (Red) 🔴
│  └─ STOP
│
└─ Pulse present and strong? → Continue to Step 3

        ↓
STEP 3: MENTAL STATUS (Age-Appropriate Behavior)
├─ No appropriate behavior/response? → Continue to Step 4
│
└─ Age-appropriate behavior/response? → DELAYED (Yellow) 🟡
   (Able to play, responds to voice, follows commands for age)

        ↓
STEP 4: AIRWAY POSITIONING (if RR normal but no behavior)
├─ After airway positioning: RR improves? → DELAYED (Yellow) 🟡
│
└─ No improvement after positioning? → IMMEDIATE (Red) 🔴

PEDIATRIC NOTES:
• Brachial (arm) or femoral (groin) pulse more reliable than radial in children
• Behavioral response more important than command-following in young children
• Airway positioning can be therapeutic intervention during triage
• Always assess quickly but completely — don't skip steps
```

## SALT (Sort, Assess, Lifesaving Interventions, Treatment/Transport)

**Use for:** Comprehensive triage with interventions | Assessment time: ~90–120 seconds

```
PHASE 1: SORT (Initial Wave Assessment)
├─ Walking toward safe area? → "Minor" group (Green)
├─ Cannot walk, needs help? → Assess individually below
└─ No movement? → Assess last

        ↓
PHASE 2: ASSESS (Vital Signs)
For each patient, check:

├─ NO BREATHING (after 10-sec airway maneuver)? → EXPECTANT ⚫
├─ No radial pulse? → IMMEDIATE 🔴
├─ Respiratory rate ≥ 30? → IMMEDIATE 🔴
├─ Altered mental status? → IMMEDIATE 🔴
├─ Non-compressible truncal hemorrhage? → EXPECTANT ⚫
│
└─ None of above? → Continue to PHASE 3

        ↓
PHASE 3: LIFESAVING INTERVENTIONS
Apply immediately:
├─ Airway positioning (head tilt/chin lift)
├─ Hemorrhage control (direct pressure, elevation, tourniquet)
├─ Decompression (if trained; tension pneumothorax)
└─ Repositioning (recovery position if safe)

Then REASSESS:
├─ Improved after intervention? → May downgrade category
└─ No change? → Maintain assigned category

        ↓
PHASE 4: TREATMENT/TRANSPORT
└─ Route to appropriate destination
```

## ESI (Emergency Severity Index)

**Use for:** ED triage for resource allocation | Assessment: 2–5 minutes

```
PRESENTATION AT ED TRIAGE

ESI-1 DECISION: Does patient require IMMEDIATE LIFE-SAVING INTERVENTION?
├─ Intubation? Resuscitation? Emergent decompression?
├─ YES → **ESI-1 (Emergent)** 🔴 → Immediate acute care bed
└─ NO → Continue to ESI-2 Decision

        ↓
ESI-2 DECISION: Is this a HIGH-RISK SITUATION?
(Confusion, shock, severe pain, respiratory distress, etc.)
├─ High-risk presenting problem?
│  └─ Chest pain, substance overdose, severe sepsis, etc.
├─ High-risk disease history?
│  └─ Immunocompromised, on anticoagulation, etc.
├─ Confusion, disorientation, change in mental status?
├─ Severe pain/distress?
│
├─ YES to any → **ESI-2 (Emergent)** 🟠 → Immediate evaluation
└─ NO → Continue to ESI-3 Decision

        ↓
ESI-3 DECISION: Does patient require MULTIPLE OR COMPLEX RESOURCES?
(Needs imaging, procedures, IV meds, etc.)
├─ CT scan, ultrasound, lab work, IV medications?
├─ Specialty consultation, urgent procedure?
├─ Observation or admission likely?
│
├─ YES → **ESI-3 (Urgent)** 🟡 → Queue for evaluation
└─ NO → Continue to ESI-4/5 Decision

        ↓
ESI-4 vs. ESI-5 DECISION: Minor vs. Least Urgent
├─ Stable vital signs, one system affected, minimal resources
├─ ESI-4: Requires ONE or TWO resources (e.g., wound repair, single lab)
│  └─ Example: Simple laceration, sore throat
├─ ESI-5: Requires essentially NO resources
│  └─ Example: Medication refill, recheck at follow-up
│
└─ Route accordingly: ESI-4 to queue; ESI-5 may divert to fast-track

ESI SUMMARY TABLE:
| Level | Priority | Intervention | Location | Example |
|-------|----------|--------------|----------|---------|
| ESI-1 | Emergent | Life-saving | Resus/ICU | Cardiac arrest |
| ESI-2 | Emergent | High-risk eval | Acute care bed | Chest pain |
| ESI-3 | Urgent | Multiple resources | ED queue | Fracture needing imaging |
| ESI-4 | Less urgent | Limited resources | Queue or fast-track | Laceration |
| ESI-5 | Least urgent | Minimal resources | Fast-track or divert | Medication refill |
```

## Critical Differentiators

### Immediate vs. Delayed
| Finding | Category |
|---------|----------|
| RR <10 or >30 | Immediate |
| No radial pulse OR CRT >2 sec | Immediate |
| Altered mental status / cannot follow commands | Immediate |
| Severe bleeding (non-compressible) | Immediate |
| **Stable vital signs + minor injury** | **Delayed** |
| **Stable vital signs + alert** | **Delayed** |

### Delayed vs. Minor
| Finding | Category |
|---------|----------|
| Serious injury (fracture, moderate bleed, deep wound) | Delayed |
| Need for splinting, assessment, intervention | Delayed |
| **Walking, alert, minimal/no injury** | **Minor** |
| **Self-ambulatory with minor complaint** | **Minor** |

### When to Over-Triage
When UNCERTAIN → Triage HIGHER (more urgent) category:
- Unsure about respiratory rate? Treat as IMMEDIATE
- Unsure about mental status? Treat as IMMEDIATE
- Unsure about perfusion? Treat as IMMEDIATE
- **Rule:** Better to over-triage (higher acuity) than under-triage

## Common Triage Errors & Corrections

| Error | Correction |
|-------|-----------|
| "Patient is walking, so definitely Minor" | Still assess all steps; abnormal vitals = higher category |
| "Respiratory rate exactly 30 = normal" | RR ≥30 = abnormal; triage as IMMEDIATE |
| "Patient seems alert, so must be okay" | Assess ALL THREE STEPS; one abnormality = abnormal category |
| "Pulse is hard to find, must be okay" | Weak/absent pulse = IMMEDIATE; assess carefully |
| "45 minutes trapped = non-salvageable" | Assess salvageability based on injuries, not entrapment time |
| "Child breathing fast, probably scared" | RR >40 for age 1–8 = IMMEDIATE; assess carefully |
