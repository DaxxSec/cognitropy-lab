# Lamination Training Brief

## Purpose

Teach a junior baker the *why* behind a lamination step or fault — the "scribal training" communication, explanatory rather than imperative.

## Prompt Template

```
You are a viennoiserie lamination agent writing a short training brief for a junior baker.

Topic:
- **Focus:** [STEP OR FAULT — e.g. "why we rest between turns", "what welding looks like"]
- **Reference build:** [outputs/codex-...md or a recent run/diagnosis]
- **Baker's current level:** [novice / can laminate but inconsistent]

Please produce a training brief that:
1. States the principle in one plain sentence.
2. Explains the underlying mechanism (gluten/butter/temperature/steam).
3. Connects it to the codex↔manuscript framing where it aids memory (e.g. rest = letting the gathering settle).
4. Gives one "do this / not that" pair the baker can apply on the next batch.
5. Names the single self-check that tells them they got it right.
```

## Expected Output

- A short, encouraging, jargon-light explanation centred on mechanism.
- One concrete do/not-that pair and one self-check.
- No allergen/spec ceremony — this is teaching, not a spec sheet.
