# Pastry Lamination Dough Technique — Reference Tables

Compact lookup data. Prose lives in `concepts.md`; procedures in `workflows.md`.

## Fold → fat-layer math

Fat layers = product of each turn's multiplier (starting from the single lock-in layer = 1).

| Notation | Fold type | Multiplier |
|---|---|---|
| `S` | Single / letter / trifold (*tour simple*) | ×3 |
| `B` | Book / double / wallet (*tour double*, *portefeuille*) | ×4 |

| Regimen | Formula | Fat layers | Typical product |
|---|---|---|---|
| `S·S·S` | 3³ | 27 | Croissant / pain au chocolat |
| `S·S·S·S` | 3⁴ | 81 | Rich Danish |
| `B·S·S` | 4·3·3 | 36 | Croissant variant (faster) |
| `B·B·S` | 4·4·3 | 48 | Puff (compromise) |
| `S·S·S·S·S·S` | 3⁶ | 729 | Classic *pâte feuilletée* |
| `B·B·B` | 4³ | 64 | Quick puff |

Total alternating leaves ≈ 2 × fat layers + 1.

## Material spec targets

| Material | Spec | Target | Note |
|---|---|---|---|
| Laminating butter | Fat % | ≥ 82% (AOP *beurre de tourage* ~84%) | Low water = clean sheeting |
| Butter | Plasticity / working temp | 13–16 °C | Bends without cracking or oozing |
| Flour (croissant) | Protein % | 11.5–12.5% | Medium-strong; structure without snapback |
| Détrempe | Dough temp at lamination | 12–15 °C | Match butter plasticity |
| Détrempe | Hydration | 50–58% | Lower for puff, higher for yeasted viennoiserie |

## Temperature & timing windows

| Stage | Window | Failure if violated |
|---|---|---|
| Lock-in | Butter 13–16 °C, dough 12–15 °C | Crack (cold) / bleed (warm) |
| Rest between turns | 20–45 min @ 3–5 °C | Snapback, smearing |
| Final proof | 24–26 °C, RH 75–80% | > melt point → butter melts out |
| Bake (initial) | 200–220 °C | Low → no steam lift; high → gum line |
| Butter melt point | ~32–35 °C | Hard ceiling for proof/handling |

## Defect → likely cause (quick triage)

| Crumb evidence | Likely cause | First fix |
|---|---|---|
| Welded / bready layers | Warm lock-in or soft butter | Re-temper to window; add rest |
| Grey / leaden streaks | Overworked, butter smeared | Gentler sheeting; chill between turns |
| Gum line at base | Under-proof / too much bottom heat | Longer proof; deck/oven adjustment |
| Butter pooling / bleed | Fat too soft / block too thick | Colder lock-in; thinner block |
| Weak / no feet, poor lift | Under-lamination / weak gluten / cool oven | Verify turn count; hotter initial bake |
| Tunnelling / blowouts | Uneven proof, trapped gas | Even proof; dock if needed |

## Paleography ↔ lamination glossary

| Manuscript term | Meaning | Lamination analogue |
|---|---|---|
| Quire / gathering | Folded stack of sheets | A turned, folded dough block |
| Bifolium | One sheet folded once | One fold's leaf pair |
| Collation formula | Notated quire structure | Notated fold regimen (`S·S·S`) |
| Diplomatic transcription | Exact copy incl. errors | Exact bake-run record incl. deviations |
| Palimpsest | Scraped, overwritten parchment | Welded layers hiding the original structure |
| Foxing / cockling | Spotting / warping damage | Butter bleed-through / block warp |
| Provenance / colophon | Origin & maker record | Ingredient batch trace & baker's signature |
| Mise-en-page | Page layout | Block geometry / lamination geometry |
