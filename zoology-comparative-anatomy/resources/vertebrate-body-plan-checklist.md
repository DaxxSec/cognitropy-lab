# Vertebrate Body-Plan Inspection Checklist

**Version:** 1.0 (2026-04-25)
**Scope:** Vertebrata; adapt per clade. Invertebrate phyla require a phylum-specific checklist (see `resources/comparative-anatomy-glossary.md` for pointers).

For every specimen × system pair, walk every line. Record `present` / `absent` / `damaged-uncertain` / `not-evaluable` / `not-applicable` for each item. Record measurements with units. Record character state if a project character set is defined.

## B. Integument

- Body covering: scales (placoid / cycloid / ctenoid / ganoid / cosmoid / keratinized) / feathers / hair / glandular / naked
- Scale / feather / hair density (if relevant): per cm²
- Color pattern: describe; photograph reference
- Pigmentation: superficial pattern + cryptic / aposematic / sexually dimorphic
- Glands: presence / type (mucous, serous, sebaceous, mammary, scent, venom, electric)
- Claws / nails / hooves / talons: present / absent / count / form
- Horns / antlers / cranial appendages: present / absent / form / measurement
- Specialized integumentary derivatives (osteoderms, scutes, baleen, etc.): list

## C. Skeletal system — Cranial

### C.1 Skull (general)
- Total length (CBL — condylobasal length): mm
- Total width (zygomatic width or equivalent): mm
- Skull shape index: ratio
- Number of cranial bones (count if possible): integer

### C.2 Dermatocranium
- Premaxilla / maxilla: present / sutures / measurements
- Nasal: present / paired / fused / measurements
- Frontal: present / paired / fused
- Parietal: present / paired / fused
- Squamosal / temporal: form / processes
- Lacrimal: present / form
- Jugal / zygomatic: present / form / arch involvement
- Vomer: present / paired / form
- Palatine: present / extent
- Pterygoid: present / form
- Quadrate (where retained): present / form

### C.3 Chondrocranium / replacement bones
- Basioccipital, exoccipital, supraoccipital: condition (separate / fused into occipital)
- Sphenoid (basisphenoid, presphenoid, alisphenoid, orbitosphenoid): condition
- Petrosal / petromastoid: presence / form
- Ethmoid: presence / form

### C.4 Splanchnocranium / visceral skeleton
- Mandible: dentary / angular / surangular / coronoid / articular — note retained vs. fused/lost
- Hyoid apparatus: form / fused or articulated
- Branchial / gill arches (where retained): count

### C.5 Dentition
- Tooth formula: I.C.P.M / upper / lower
- Tooth replacement: monophyodont / diphyodont / polyphyodont
- Tooth attachment: acrodont / pleurodont / thecodont
- Tooth class differentiation: heterodont / homodont
- Cusp pattern (mammals): tribosphenic terminology (paraconid, protoconid, metaconid, paracone, protocone, metacone, hypocone, etc.)
- Wear stage: unworn / light / moderate / heavy / molarized

### C.6 Cranial foramina (transmitting nerves/vessels)
- Foramen magnum: shape / area / orientation
- Optic foramen / canal: present / form
- Foramen rotundum (V2): present / form
- Foramen ovale (V3): present / form
- Internal acoustic meatus: present / form
- Hypoglossal foramen: present / count
- Carotid canal / foramen: present / form
- Other clade-specific foramina (cite literature)

## D. Skeletal system — Postcranial axial

### D.1 Vertebral column
- Total count (cervical / thoracic / lumbar / sacral / caudal): integers
- Vertebral type: amphicoelous / opisthocoelous / procoelous / heterocoelous / acoelous
- Cervical specializations: atlas / axis form
- Sacral fusion (sacrum): vertebral count fused
- Caudal series: free count vs. pygostyle / synsacrum / urostyle as applicable

### D.2 Ribs
- Rib count (paired): integer
- Rib type: bicapitate / unicapitate
- Rib morphology: floating / true / false (mammalian); uncinate processes (sauropsids)

### D.3 Sternum
- Form: cartilaginous / partially ossified / fully ossified
- Carina / keel: present / absent / size (avian / chiropteran)

## E. Skeletal system — Appendicular

### E.1 Pectoral girdle
- Scapula: form, length, glenoid orientation
- Coracoid: present / absent / fused / form
- Clavicle: present / absent / form
- Interclavicle (where retained): present / form
- Suprascapular / suprascapula: present / cartilaginous

### E.2 Forelimb
- Humerus: length / proximal articulation / distal articulation / deltopectoral crest
- Radius / ulna: lengths / olecranon process
- Carpal count + arrangement
- Manus: digit count / phalangeal formula (e.g. 2-3-3-3-3 for therian mammals); claw / nail / hoof terminus

### E.3 Pelvic girdle
- Ilium / ischium / pubis: fused / separate / form
- Acetabulum: closed / open / form
- Pelvic-girdle attachment to vertebral column: number of sacral contacts

### E.4 Hindlimb
- Femur: length / proximal articulation / trochanter pattern
- Tibia / fibula: lengths / fusion (tibiotarsus in birds)
- Tarsal count + arrangement (calcaneum, astragalus, navicular, cuboid, cuneiforms)
- Pes: digit count / phalangeal formula; claw / nail / hoof terminus

## F. Muscular system

- Major epaxial groups (epaxial dorsal trunk musculature)
- Major hypaxial groups
- Appendicular muscles — pectoral and pelvic (origin / insertion / innervation if known)
- Specialized muscles (clade-specific, e.g. mammalian diaphragm; avian flight muscles): presence + form
- Cutaneous muscles (panniculus carnosus): present / absent / regional development

## G. Cardiovascular system

- Heart chamber count: 2 / 3 / 4
- Atrial / ventricular septation
- Aortic arch retention pattern (1-6 system; mammals retain L4, birds retain R4, etc.)
- Major arteries: aortic arch direction (L / R / both); systemic / pulmonary separation
- Major veins: anterior / posterior caval system; renal portal (where retained)
- Lymphatic system noted if examined

## H. Respiratory system

- Gills: present / absent / count / type (internal / external)
- Lungs: paired / unequal / absent
- Lung structure: alveolar (mammalian) / faveolar / parabronchial (avian) / sac-divided
- Air sacs (avian): count / location
- Trachea / bronchi pattern

## I. Digestive system

- Oral cavity features: tongue / teeth / palate (primary / secondary)
- Pharynx: gill-bearing / not
- Esophagus: length / specializations (crop in birds)
- Stomach: simple / chambered (ruminant / pseudoruminant / monogastric); gizzard if present
- Intestine: length (relative to body length); spiral valve (where retained); cecum (count + size)
- Cloaca / anus: cloaca retained / urogenital + alimentary separated
- Liver: lobe count
- Pancreas: discrete / diffuse
- Spleen: present / form

## J. Urogenital system

- Kidney type: pronephros / mesonephros / metanephros (depending on developmental stage and clade)
- Kidney shape / lobation
- Ureter / Wolffian duct / Müllerian duct retention
- Gonads: paired / unpaired / position (intra-abdominal / scrotal / peritoneal)
- Reproductive ducts: oviduct, uterus form / horns
- Cloaca features: monotreme cloaca / hemipenes / penis / phallic morphology

## K. Nervous system

- Brain regional development: olfactory bulb / cerebrum (with relative gyrification) / midbrain optic tecta / cerebellum / medulla
- Encephalization: brain mass relative to body mass (brain mass — g; body mass — g)
- Spinal cord: segmental enlargements (cervical, lumbosacral)
- Cranial nerves: count of pairs identified (mammals 12; lower vertebrates often 10 — note clade convention)
- Peripheral nerves of interest

## L. Sensory systems

- Eye: present / absent / form / specializations (tapetum, eyelid types, nictitating membrane)
- Ear: outer / middle / inner; auditory ossicle count (1 in non-mammalian tetrapods, 3 in mammals)
- Lateral line system (where retained): present / absent / pattern
- Olfactory: turbinates / vomeronasal organ presence
- Electroreception: present / absent / receptor type (where applicable)
- Magnetoreception: documented? (literature reference)

## M. Final aggregation

After walking every system that applies to the taxon:
- Note any features that diverge from the modal expectation for the clade.
- Flag any structures that were not-evaluable due to specimen condition.
- Cross-reference: which inspection records apply to this specimen if multiple system files were generated.
- If the project has a character set, all coded character states should now be filled in (or `?` / `-`).

## Adaptations per clade — reminders

- **Mammals:** include mammary glands, hair, three middle-ear ossicles, four-chambered heart, secondary palate, mandibular dentary-only, alveolar lung.
- **Birds (Aves):** include feathers, beak (rhamphotheca), keeled sternum, fused tarsometatarsus and tibiotarsus, parabronchial lung, air sacs, pygostyle, R4 aortic arch, three middle-ear ossicles? — only one (columella).
- **Crocodilians:** include four-chambered heart with foramen of Panizza, secondary palate (different ontogeny than mammals), single columella.
- **Squamates:** include hemipenes, lingual structure (forked / not), specialized integument.
- **Testudines:** include shell (carapace + plastron), beak (rhamphotheca), reduced/absent dentition.
- **Lissamphibia:** include glandular skin, larval gill stage (anurans / urodeles), incomplete septation, kidney = mesonephros in adult.
- **Actinopterygii:** include scales (cycloid / ctenoid / ganoid / placoid in chondrichthyans), gills, swim bladder vs. lung condition, heterocercal vs. homocercal vs. diphycercal tail.
- **Chondrichthyes:** include placoid scales, cartilaginous skeleton, claspers (males), spiral valve intestine, lateral line.
- **Agnatha (Cyclostomata):** include lack of jaws, lack of paired fins, branchial basket, single nostril (cyclostomes), notochord retained.

For invertebrate phyla, replace this checklist with the phylum's standard reference (Brusca & Brusca; Ruppert et al.; phylum monograph). Do not mechanically apply this vertebrate-centric list.
