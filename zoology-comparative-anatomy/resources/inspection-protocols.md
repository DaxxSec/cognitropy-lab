# Inspection Protocols — Per-System Procedure Reference

Detailed protocols for examining each major body system. These extend the body-plan checklist (`vertebrate-body-plan-checklist.md`) with operational guidance: what to look at, in what order, with what instruments, with what risk-of-error notes.

The agent should consult the relevant section before running `/inspect` on a specimen for that system.

---

## Skull (Osteology)

### Order of operations
1. Photograph or visualize from six standard views: dorsal, ventral, left lateral, right lateral, anterior, posterior. Note any view that's missing.
2. Take **CBL** (condylobasal length): from the anterior tip of the premaxilla to the posterior surface of the occipital condyles.
3. Take **zygomatic width**: maximum width across zygomatic arches.
4. Take **braincase width**: maximum braincase width perpendicular to the long axis.
5. Walk the dermatocranium suture-by-suture: identify each bone, note suture state (open / partially fused / fully fused / obliterated).
6. Walk replacement bones (basisphenoid, alisphenoid, etc.).
7. Walk splanchnocranial elements.
8. Inventory cranial foramina from ventral view: each foramen named, position, what passes through (cite reference).
9. Dental inspection: record formula, occlusion, wear stage, any anomalies.

### Common error sources
- Suture state misjudged on cleared specimens (post-mortem opening looks like patent suture). Cross-check with X-ray or CT if available.
- Foramen identification confused on damaged ventral surface — when in doubt, photograph and flag rather than guessing.
- Tooth class assignment in re-erupted or worn teeth: use position + pre-eruption tooth-bud morphology if available.

### Standards to cite
TA2 for cross-vertebrate; NAV for non-human mammal-specific; clade-specific atlas where relevant (e.g. Smith 1996 for early tetrapod skulls; Goodrich 1930 for Mesozoic forms).

## Postcranial Axial Skeleton

### Order of operations
1. Vertebra-by-vertebra walk from atlas to terminal caudal. Photograph each in lateral + cranial-end views.
2. Count cervical / thoracic / lumbar / sacral / caudal series. (Definition of regional boundaries differs by clade — anchor in the clade-standard convention.)
3. Type each vertebra (amphicoelous / opisthocoelous / procoelous / heterocoelous / acoelous) based on centrum end profiles.
4. Ribs: count pairs, identify true / false / floating, measure rib length (longest preserved).
5. Sternum (where present): note ossification state, keel size, segment count.

### Common error sources
- Boundary between thoracic and lumbar is conventional, not absolute. Define by rib articulation: thoracic = rib-bearing; lumbar = rib-free in mammals. Other clades define differently (e.g. archosaurs).
- Sacral count: include vertebrae fully fused into sacrum + those contacting ilium via rib element. Conventions vary; cite the convention.
- Caudal vertebrae: pygostyle (birds) and urostyle (anurans) are fused; count pre-fusion segments separately if osteologically resolvable.

## Appendicular Skeleton

### Order of operations
1. Pectoral girdle: scapula, coracoid (where retained), clavicle, interclavicle (where retained), suprascapula. Photograph each in standard views.
2. Forelimb: humerus → radius/ulna → carpals (count + arrangement) → manus (digit count + phalangeal formula + terminal element type).
3. Pelvic girdle: ilium / ischium / pubis. Note acetabulum (closed / open). Count sacral contacts.
4. Hindlimb: femur → tibia/fibula (note fusion to tibiotarsus in birds) → tarsals → pes (digit count + phalangeal formula + terminal element type).

### Linear measurements
For each long bone: maximum length parallel to long axis, midshaft circumference (if intact), proximal articular width, distal articular width.

### Common error sources
- Phalangeal formula numbering: count from medial (digit I) to lateral (digit V). Polydactyly / oligodactyly noted as deviations from clade-standard formula.
- Carpal / tarsal fusion confusing in juveniles (some elements not yet fused). Age class affects character coding — record age class explicitly.
- "Functional digits" vs. "anatomical digits" — coding should be anatomical. Ungulate hooves arise from digit III (perissodactyl) or fused III+IV (artiodactyl).

## Cardiovascular System

### Order of operations
1. Heart in situ: orientation, position relative to ribs/sternum.
2. Heart removed: weigh (g), photograph, identify chambers from external view.
3. Section heart along plane that exposes atria + ventricles + septa. Document septum completeness.
4. Identify aortic arch direction (L / R / both).
5. Trace aortic-arch retention pattern (arches 1-6, identify which retained as adult derivatives).
6. Major systemic arteries traced: subclavian, common carotid → external/internal, abdominal aortic branches.
7. Major veins traced: anterior cava (L+R or single), posterior cava, hepatic portal, renal portal (where retained).

### Common error sources
- Chamber count: amphibians have a single ventricle but partial septation. Don't reflexively call this "3-chambered" without stating the criterion (3 anatomical chambers vs. 2-with-partial-septation is the same condition described differently).
- Aortic arch retention: birds (R4) and mammals (L4) is the textbook. Check actual specimen, don't assume.
- Pulmonary circulation in lungfish vs. amphibians: both have anatomical pulmonary circuits but with different completeness of separation.

## Respiratory System

### Order of operations
1. External: nostrils / nares / spiracles / blowhole. Position, count, paired/unpaired.
2. Pharyngeal: branchial / pharyngeal arches retained (gills, gill rakers, pseudobranch).
3. Lungs (where present): paired / unpaired / lobation / lobar count. Note clade-standard variations.
4. Internal lung structure: alveolar / faveolar / parabronchial.
5. Air sacs (avian): identify by position (interclavicular, anterior thoracic, posterior thoracic, abdominal, cervical).
6. Trachea + bronchi: tracheal ring count if relevant; bronchial branching pattern.

### Common error sources
- Lobation: mammalian lungs typically asymmetric (heart accommodation); 3 lobes right, 2 lobes left in human; varies across mammals.
- Avian lung is non-distensible — saccular airflow occurs via air sacs. Don't describe as "two lungs" without the parabronchial + air-sac context.
- Cetacean / sirenian aquatic adaptations: respiratory anatomy specialized; consult cetacean-specific reference (Mead & Fordyce 2009 for cetaceans).

## Digestive System

### Order of operations
1. Oral cavity: tongue (form, attachment, lingual papillae if relevant), palate (primary / secondary), dental occlusion.
2. Pharynx: gill-bearing / not.
3. Esophagus: length (relative to body length), specializations (crop in some birds).
4. Stomach: simple / chambered. For ruminants: identify rumen / reticulum / omasum / abomasum.
5. Intestine: total length (mm), small + large intestine if differentiable, spiral valve (chondrichthyans / non-teleost actinopterygians retain).
6. Cecum / cecal apparatus: count, length.
7. Liver: lobe count, gall bladder presence/absence (some clades lack: e.g. cetaceans, equids).
8. Pancreas: discrete / diffuse; multiple lobes if relevant.
9. Cloaca / anus: cloaca retained / urogenital + alimentary separated.

### Common error sources
- Ruminant chamber identification: differential preservation of muscular layers can confound chamber boundaries. Use mucosal-surface morphology as primary indicator.
- Intestinal length: contracted post-mortem; measure relaxed if possible, or note state.
- Cecal differentiation in monogastric mammals — herbivorous monogastrics (e.g. equids, lagomorphs) have enlarged ceca; carnivores typically have minimal cecum.

## Urogenital System

### Order of operations
1. Kidney: identify type (pronephros / mesonephros / metanephros) appropriate to developmental stage. Adult mammals = metanephric; adult amphibians = mesonephric.
2. Kidney shape, lobation, position relative to vertebrae.
3. Ureters: course from kidney to bladder / cloaca.
4. Bladder (where retained): present / absent (birds have minimal bladder).
5. Gonads: position (intra-abdominal / scrotal / peritoneal), shape, size.
6. Reproductive ducts: male — Wolffian-derived ductus deferens; female — Müllerian-derived oviduct / uterus / vagina. Note clade-standard form.
7. Phallic morphology (male): hemipenes (squamates), penis form (mammals), phallus (paleognath birds + waterfowl + ratites; absent in most neognaths).
8. Cloaca features: monotreme cloaca (alimentary + urogenital share opening); divided cloaca conditions.

### Common error sources
- Kidney type misjudged in juvenile / immature specimens — adult anatomy is the standard for character coding unless the project explicitly studies development.
- Phallic morphology: many bird species lack a phallus despite the rhetoric "all male animals have one." Be precise.
- Reproductive duct fusion in mammals (uterus simplex / bicornis / duplex) is a major comparative character.

## Nervous System

### Order of operations
1. Brain in situ: photograph from dorsal + ventral + lateral (both sides) views.
2. Brain weight (g) + body weight (g) for encephalization measure.
3. Identify brain regions: olfactory bulbs, cerebrum (note gyrification in mammals), midbrain optic tecta, cerebellum (folds: mammalian foliation), medulla.
4. Cranial nerves: 12 pairs in mammals (CN I-XII); typically 10 in lower vertebrates (combined arrangements vary). Identify exit foramina, then trace where possible.
5. Spinal cord: segmental enlargements (cervical for forelimb, lumbosacral for hindlimb in tetrapods), terminal filum.

### Common error sources
- Brain mass measurement requires careful removal — fixed specimens have shrunk; live brain mass differs. Cite preservation and measurement context.
- Cranial nerve count convention: mammalian-style 12-pair count vs. older 10-pair count of comparative anatomy literature. Use one convention; cross-reference both if unclear.
- Optic tectum vs. superior colliculus: same structure, different names by clade. Don't double-count.

## Sensory Systems

### Order of operations
1. Eye: photograph in situ + dissected. Note size relative to skull (orbit area), pupil shape (round / vertical slit / horizontal slit / w-shape — informative on ecological niche).
2. Eyelids: upper / lower / nictitating membrane state.
3. Tapetum lucidum present/absent (nocturnal/aquatic indicator).
4. Ear: outer ear (pinnae + external auditory meatus) — present in mammals only. Middle ear: ossicle count (1 in non-mammalian tetrapods = columella; 3 in mammals = malleus, incus, stapes). Inner ear: cochlear coil count (mammals = up to 4+ turns; non-mammalian = simpler).
5. Lateral line system: in fishes and aquatic anamniotes. Photograph the canal pattern. Note any neuromast aggregations.
6. Olfactory: turbinates (count + form in mammals), vomeronasal organ (presence + form — absent in many primates and birds).
7. Electroreception: ampullae of Lorenzini in chondrichthyans; passive + active electroreception in monotremes.
8. Magnetoreception: documented in birds, sea turtles, salmonids. Cite literature; not directly observable in dissection.

### Common error sources
- Pupil shape determined in life or fixed-fresh specimens; preservation can deform.
- Cochlear coil count: requires clearing/CT or careful microdissection.
- Lateral line trace: post-mortem damage or scale loss can obscure pattern.

## When the Specimen Doesn't Fit a Single Protocol

Many specimens require multi-protocol passes (e.g. a dolphin pelvis is a vestigial element best examined under a "vestigial structure" lens not covered explicitly here). When the standard protocols are inadequate, document the deviation, propose an extended protocol, and have the user approve before continuing.
