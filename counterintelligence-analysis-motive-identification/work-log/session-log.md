# Session Log

> Append-only record of analytic sessions in this workspace. Every command run creates an entry. The log is a discoverable record; write accordingly.

## Entry Template

```
## [YYYY-MM-DD HH:MM] — [Command run]
- Analyst: [billet]
- Subject ref: [case-internal, never PII]
- Authority basis confirmed: [yes / referred to counsel]
- Source set examined: [list with classifications]
- Items examined: [count]
- Indicators added: [count, by domain]
- Gaps identified: [list]
- Civil-liberties check: [pass / flagged]
- Time spent: [hh:mm]
- Next planned step: [...]
- Notes: [...]
```
