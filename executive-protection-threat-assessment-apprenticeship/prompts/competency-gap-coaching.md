# Competency-Gap Coaching

## Purpose

Turn a progression review's weakest-EPA finding into a concrete deliberate-practice coaching plan for an apprentice agent — the bridge between "you're weak here" and "do exactly this next."

## Prompt Template

```
You are a protective-program training officer coaching a named apprentice agent.

I have the following inputs:

- **Apprentice + Dreyfus stage:** [NAME, ROLE]
- **Weakest EPA + current entrustment:** [EPA-N, LEVEL 1–5]
- **Evidence of the gap:** [SPECIFIC OBSERVATIONS FROM SIGNOFFS/AARs]
- **Available assignments this cycle:** [REAL OR SCENARIO TASKS]
- **Context:** [TIME AVAILABLE, SUPERVISION CAPACITY]

Please:
1. Diagnose the *root* of the gap (knowledge, reasoning, judgement, or performance under stress).
2. Set 1–2 deliberate-practice objectives at the edge of ability (one entrustment level up).
3. Specify the scaffolding to provide now and the scaffolding to fade next.
4. Define the observable that will show the gap is closing and the WBA that confirms it.
```

## Expected Output

- A coaching plan: root-cause diagnosis, 1–2 edge objectives, scaffolding-fade plan, and the closing observable/WBA.
- Targeted at the weakest competency, not a generic "do more reps."
- Respects the entrustment ceiling — no live-operation stretch beyond the proven level.
