# Prompt Template: Build Planning & Upgrade Roadmap

Use this template when you want Claude to help plan the next phase of your build — what to do, in what order, and why.

---

## Full Build Roadmap Template

```
I want to plan the next phase of upgrades for my build.

**Current state:** [Brief description of what's on the car now — or say "see project.md"]
**Current power:** [WHP / WTQ on what fuel at what boost]
**Target power:** [WHP / WTQ on what fuel]
**Budget available:** [Approximate — or confirm from constraints.md]
**Timeline:** [When do you want this done by?]
**Primary use:** [Street / Track / Drag / Mixed]

**Current limiting factor (if known):**
[What do you think is holding back more power? Or say "not sure"]

**Mods I'm considering (if any):**
- [Part 1]
- [Part 2]

**Constraints:**
- [Must keep street legal / emissions compliant]
- [Prefer to retain OEM ECU]
- [Other constraints from your context]

Please give me a recommended upgrade path in logical order — what to do first,
why that order matters, and what each step unlocks. Flag if any step requires
a retune or has safety implications.
```

## Quick "What Should I Do Next?" Template

```
My [VEHICLE] is making [CURRENT POWER] on [FUEL] with [CURRENT SETUP].
I want to get to [TARGET POWER]. Budget is roughly [$XXX].

What's the most cost-effective next step? What's the limiting factor right now?
```

---

## "Will This Work?" Template

```
I'm considering adding [SPECIFIC PART] to my [VEHICLE].

Current setup: [brief summary or see project.md]
Goal: [What I'm hoping this achieves]

Questions:
1. Will this physically fit my setup?
2. Does it require supporting mods?
3. Will I need a retune?
4. Is this the right move given my build goals?

Budget for this part: [$XX]
```

## Engine Build Spec Planning Template

```
I'm planning a [full engine rebuild / bottom end build / head build] for my [ENGINE].
Target power: [WHP on FUEL]
Primary use: [Street / Track / Drag]
Budget for the build: [$XXXX]

Can you suggest a parts spec that hits my targets while staying within budget?
Include: pistons, rods, head gasket, studs, seals, and any head work recommendations.
Flag any safety margins I should be aware of with this spec.
```
