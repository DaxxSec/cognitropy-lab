# Prompt Template: Session Wrap-Up & Work Log Entry

Use after any tuning session, dyno run, or garage session to document what happened and capture the next steps.

---

```
Dyno session wrap-up for [DATE].

**Location / Dyno:** [shop name, dyno type]
**Fuel:** [octane, ethanol]
**Conditions:** [ambient temp, humidity if known]
**Number of pulls:** [X]

**Results:**
- Best pull: [WHP] / [WTQ] @ [RPM] on [fuel]
- Previous best for comparison: [WHP / WTQ from last session]
- Boost achieved: [X psi peak / X psi average]
- AFR at WOT: [range]
- Any knock events: [yes — describe / no]
- Any other anomalies: [describe or none]

**Tune changes made during session:**
[List changes or "no changes — baseline only"]

**What went well:**
[Notes]

**What to address next session:**
[Notes]

Please create a work-log entry and update the power numbers in project.md.
```
