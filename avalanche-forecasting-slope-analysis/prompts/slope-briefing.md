# Prompt: Guide / Patrol Slope Briefing

Produce a 90-second verbal briefing script for a guide or patrol team heading into a named slope today.

Required elements:
1. Objective + slope basics (aspect, elevation, angle)
2. Today's problem types active on this feature
3. Specific evidence from the last 24 h (pit, obs, weather)
4. Two triggers that would prompt a turnback
5. FACETS trap most likely for this party today
6. Mitigation plan if "go" decision

Plain English. No jargon without a parenthetical gloss.
