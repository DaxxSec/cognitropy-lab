# Prompt: Bulletin Tone & Language Audit

Review a draft forecast bulletin and flag any phrasing that:

1. Uses the word "safe" or synonyms ("totally safe," "no danger")
2. Downgrades a persistent weak layer problem without explicit tracker evidence of healing
3. Mixes avalanche problem types within a single paragraph without a clean transition
4. Cites InfoEx professional-exchange observations without a [PRO] redaction tag
5. Uses absolute tense ("no avalanches will occur") instead of probabilistic ("unlikely, though locally possible")
6. Omits a confidence statement

Return findings as a bulleted list with line / sentence citations and proposed rewrites.
