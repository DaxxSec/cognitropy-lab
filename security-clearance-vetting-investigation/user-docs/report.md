# Reports (index)

This folder holds polished, human-readable deliverables produced by the agent:
- `subjects/SUBJ-NNNN/adjudicative-memo-YYYYMMDD.md` — adjudicative drafts (for human review).
- `subjects/SUBJ-NNNN/reinvest-plan-YYYYMMDD.md` — condition-based reinvestigation plans.
- `caseload-review-YYYYMMDD.md` — periodic batch reports.

Every artifact in here carries the standard disclaimer:
> This artifact is a reasoning aid prepared by an LLM agent. Nothing here is an adjudicative determination. Due process protections for the subject are preserved and the cognizant adjudicative authority retains sole responsibility for decisions.
